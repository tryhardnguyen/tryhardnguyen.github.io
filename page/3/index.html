Enable Firewall Logs

Nguyen Nguyen published on 2023-05-08 included in Homelabs Logs
Enable Firewall Logs First, let go to the search menu and search up “Group Policy”. Click on “Edit Group Policy”. Expand Windows Settings -> Expand Security Settings -> Expand Windows Defender Firewall with Advanced Security -> Click on Windows Defender Firewall Click on “Windows Defender Firewall Properties” You should see three profile: Domain Profile, Private Profile, Public Profile. For the Domain profile: Firewall state: On Inbound Connections: Block (Default) Outbound connections: Allow (Default) For Private Profile: Firewall state: On

Enable Power Shell Logging

Nguyen Nguyen published on 2023-05-08 included in Homelabs Logs
Introduction The purpose of the post is to show how to enable logging for powershell on a Windows Client. Let’s start Start Menu -> Gpedit -> Click on Edit Group Policy Then open Computer Configuration -> Administrative Template -> Windows Components -> Find “Windows Powershell” -> Turn on PowerShell Transcription -> Enabled -> Click on Include invocation headers Click Apply. After that, find Turn on Powershell Script Block Logging Click on enabled and click apply.
